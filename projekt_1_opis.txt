TYTUŁ:
Lunatic Shop

TAGI:
Discord Ecosystem / UI/UX / Full-Stack

WPROWADZENIE:
An advanced e-commerce ecosystem project built on the Discord platform, combining a unique information architecture (UI/UX) with a proprietary backend system (dedicated bot). The project fully automates the user journey – from secure verification, through ticket generation and payment processing, to advanced action logging and real-time financial analytics.

SEKCJA_1_TYTUŁ:
Proprietary Backend Solution (Dedicated Bot)

SEKCJA_1_OPIS:
The heart of the project is a custom bot, built from scratch to eliminate external, unreliable solutions and ensure full control over user data.

SEKCJA_1_PUNKTY:
- Automated Sales Process: The bot fully handles the purchasing system. It generates private transactional channels (#stwórz-ticket), manages the product database (#wyszukaj-produkt), and processes the discount raffle logic (#losuj-zniżkę).
- Live Data Engine (Statistics): A custom script dynamically counts and updates server statistics in real time (number of customers, verified users, and total financial turnover in PLN visible in the Staty! section).
- Financial & Settlement Module: The bot features a built-in settlement system for sellers (#rozliczenia) and automatically generates transcripts for every closed transaction (#transcript), ensuring full transparency and security.
- Advanced Logging System (Audit): All critical actions (logins, purchases, order status updates in the "w realizacji" section) are processed by the bot and routed to secured administrative channels.

SEKCJA_2_TYTUŁ:
Interface Design (UI) & Experience Engineering (UX)

SEKCJA_2_PUNKTY:
- Cohesive Visual Identity (Cyberpunk/Dark Mode): The interface is designed around a dark purple aesthetic with neon accents acting as status indicators (e.g., green checkmarks for verification and transaction legitimacy).
- Clear Text UI: Utilizes custom typography and a cohesive system of structural cursors (\, ›), which dramatically improves readability (text scannability) and eliminates visual clutter.
- Cognitive Load Reduction: Implementation of an advanced permissions system. Users only see sections relevant to their current step (e.g., the client zone activates only after a successful purchase), optimizing the overall User Journey.

SEKCJA_3_TYTUŁ:
Technical Specification

SEKCJA_3_PUNKTY:
- Role in the project: Full-Stack Discord Developer / UI/UX Designer
- Backend: Custom Discord bot (API handling, ticket system, data logging, payment integrations).
- Frontend/UI: Advanced Discord channel architecture, custom formatting, proprietary message layouts / embeds.
